Tạm thời tính chay:

  • Waiting Time (Thời gian chờ) = Turnaround time - Burst time (Thời gian chạy)
  • Response Time (Thời gian phản hồi) = Time at which the process gets the CPU for the first time - Arrival time
  • Turnaround time (Thời gian hoàn thành) = Burst time + Waiting time = Exit time - Arrival time
